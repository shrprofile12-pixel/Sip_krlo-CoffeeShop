# SipKrlo Coffee

## Introduction
SipKrlo Coffee is a modern, visually appealing website designed to showcase the menu of cakes, coffees, shakes, and ice-creams.  
It features a **responsive layout**, **parallax scrolling**, **hero section with overlay**, **GSAP animations**, and a **mobile-friendly sidebar navigation**.  
The website aims to provide an interactive and smooth experience for users while browsing the menu.


## Technologies Used
- **HTML5** – Structure of the website  
- **CSS3 / Flexbox / Media Queries** – Styling and responsive design  
- **JavaScript** – Hamburger menu toggle, sidebar functionality, and interactive components  
- **Font Awesome** – Social media and icon fonts  
- **Google Fonts** – Custom typography (Poppins, Anton, Poetsen One)  


## What I Learned
- How to create **responsive navigation bars** with hamburger menus for mobile devices  
- How to structure **flexbox-based grids** for menu items  
- How to use **CSS transitions and hover effects** for interactivity  
- Integrating **external fonts and icons** for better UI  
- How to **structure a modern website** with header, hero, menu grid, and footer
