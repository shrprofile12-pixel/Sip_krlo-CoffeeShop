<h1 style="color:red;">SipKrlo Coffee ☕</h1>

<h2 style="color:red;">Introduction</h2>
SipKrlo Coffee is a modern, visually appealing website designed to showcase the menu of cakes, coffees, shakes, and ice-creams.  
It features a <strong>responsive layout</strong>, <strong>parallax scrolling</strong>, <strong>hero section with overlay</strong>, <strong>GSAP animations</strong>, and a <strong>mobile-friendly sidebar navigation</strong>.  
The website aims to provide an interactive and smooth experience for users while browsing the menu.

<h2 style="color:red;">Technologies Used</h2>
- <strong>HTML5</strong> – Structure of the website  
- <strong>CSS3 / Flexbox / Media Queries</strong> – Styling and responsive design  
- <strong>JavaScript</strong> – Hamburger menu toggle, sidebar functionality, and interactive components  
- <strong>Font Awesome</strong> – Social media and icon fonts  
- <strong>Google Fonts</strong> – Custom typography (Poppins, Anton, Poetsen One)  

<h2 style="color:red;">What I Learned</h2>
- How to create <strong>responsive navigation bars</strong> with hamburger menus for mobile devices  
- How to structure <strong>flexbox-based grids</strong> for menu items  
- How to use <strong>CSS transitions and hover effects</strong> for interactivity  
- Integrating <strong>external fonts and icons</strong> for better UI  
- How to <strong>structure a modern website</strong> with header, hero, menu grid, and footer
